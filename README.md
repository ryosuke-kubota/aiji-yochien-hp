# 愛児幼稚園 HTML Prototype

このフォルダは、提供ワイヤーフレームを静的HTMLで再現したトップページです。

- `index.html`: トップページ
- `about.html`: 園の紹介ワイヤー
- `concept.html`: 教育についてワイヤー
- `annual.html`: 年間行事ワイヤー
- `schedule.html`: 園での生活ワイヤー
- `guide.html`: 入園のご案内ワイヤー
- `design-system.html`: 下層ページ制作用のトークン・部品見本
- `style.css`: 色、余白、角丸、ボタン、カード、写真マスクなどの共通スタイル
- `script.js`: SPメニュー、重要なお知らせ切り替え、トップへ戻る操作
- `assets/images/`: AI生成した写真・イラスト素材

HTMLファイルをそのままブラウザで開いて確認できます。

下層ワイヤーは、参考サイトのページ構成をもとに、愛児幼稚園用の仮原稿・導線へ再構成しています。
